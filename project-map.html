构建主路径（起点 → 终点）

pnpm build → package.json 读取脚本 → tsc 类型检查 + vite 打包 → 从 index.html 进入，解析 main.tsx → App.tsx → 所有组件和 lib → 最终输出 dist/

运行主路径（起点 → 终点）

浏览器 请求 index.html → 加载 main.tsx → 挂载 App.tsx → 初始化状态并渲染组件 → 调用 lib 把 Markdown 转成主题化 HTML → PreviewPanel 呈现 → 用户看到页面并交互
